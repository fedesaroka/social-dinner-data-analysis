# 🍽️ La Mosca – Social Dinner Data Analysis

Proyecto personal de análisis de datos basado en el seguimiento de cenas semanales entre un grupo de amigos.

El objetivo es analizar comportamientos, patrones de asistencia y dinámicas sociales a partir de datos recolectados a lo largo del tiempo.

---

## 🌐 Aplicación en producción

La aplicación está disponible online en:

👉 [https://lamosca.pythonanywhere.com/](url)

Permite registrar nuevas cenas, cargar asistencia y almacenar los datos automáticamente en Google Sheets.

No requiere instalación, se puede acceder directamente desde el navegador.

Los cambios no se van a ver guardados en caso de ingresar como invitado.

---

## 🚀 Funcionalidades del proyecto

* Recolección y estructuración de datos de:

  * Asistencia
  * Comida
  * Precios
  * Ubicación
* Integración de múltiples fuentes de datos (Google Sheets y APIs externas)
* Cálculo de métricas clave:

  * Asistencia
  * Frecuencia
  * Costos
  * Distancias recorridas
* Generación de visualizaciones para detectar patrones y tendencias
