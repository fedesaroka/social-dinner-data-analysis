# 📊 Datos y análisis por año

Esta carpeta contiene los datos y análisis de las cenas organizados por año.

---

## 📁 Estructura

* `2024/`: datos y análisis correspondientes al año 2024
* `2025/`: datos y análisis correspondientes al año 2025

---

## 📌 Contenido

Dentro de cada carpeta se pueden encontrar:

* Datos de asistencia por participante
* Información de cada cena (comida, precio, casa, etc.)
* Archivos utilizados para análisis exploratorio
* Datos preparados para visualización en herramientas como Power BI o Tableau

---

## 🎯 Objetivo

Organizar la información de manera cronológica para facilitar:

* El análisis de evolución del grupo en el tiempo
* La comparación entre distintos años
* La construcción de visualizaciones y dashboards
